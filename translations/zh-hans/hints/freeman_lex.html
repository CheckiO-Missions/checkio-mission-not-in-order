抓住一个想法💡！创建给定data 的有序版本，并比较具有相同索引的元素。计算不相等的元素。

好的，我应该从什么开始呢？

要得到一个新的列表，它是其他列表的排序变体，请使用内置的sorted() 函数。默认值为升序，因此无需更改任何其他参数。 

data1 = sorted(data)
下一步做什么？

现在您需要同时遍历两个列表，逐个索引。实现这一任务的完美工具是内置的zip() 函数。它创建了一个生成器，生成具有相同索引的值。注意，您可以使用任意数量的迭代参数，而不仅仅是两个。 

zip(data, sorted(data))

如果你不打算多次使用变量，就没有必要给变量赋值。现在使用for 循环遍历值对。 

for i, j in zip(data, sorted(data))
解释一下，我应该如何处理这些值对？

您只需对它们进行比较，然后得到它们不等式的逻辑值。 

i!= j for i, j in zip(data, sorted(data))
如何计算不相等元素的个数？

Python 可以将逻辑值解释为整数：True 等于1 ，False 等于0 。因此，每个不相等元素的情况都会得到i!= j => True => 1 。你只需要使用前面提示的表达式作为内置函数sum() 的参数。它累加 1-s，而 0-s 没有影响。 

sum(i!= j for i, j in zip(data, sorted(data)))
那么，我的最终解决方案应该是怎样的呢？

您只需返回一个最终值，并将代码封装为函数表达式。强烈建议对输入和输出数据使用类型提示。

以下是整个解决方案：

def not_order(data: list[int]) -> int:

    return sum(i!= j for i, j in zip(data, sorted(data)))

解决方案是相同的，但使用匿名lambda 函数、operator 标准模块和内置map() 函数。 

from operator import ne


not_order = lambda data: sum(map(ne, data, sorted(data)))
