Fangen Sie eine Idee 💡! Erstellen Sie eine geordnete Version der gegebenen data und vergleichen Sie Elemente mit demselben Index. Zählen Sie die Elemente, die nicht gleich sind.

Ok, womit soll ich anfangen?

Um eine neue Liste zu erhalten, die eine sortierte Variante einer anderen Liste ist, verwenden Sie eine eingebaute sorted() Funktion. Die aufsteigende Reihenfolge nach Wert ist voreingestellt, so dass Sie kein anderes Argument ändern müssen. 

data1 = sorted(data)
Was ist als nächstes zu tun?

Jetzt müssen Sie über beide Listen gleichzeitig iterieren, Index für Index. Das perfekte Werkzeug für diese Aufgabe ist die integrierte Funktion zip(). Sie erzeugt einen Generator, der Werte mit demselben Index liefert. Beachten Sie, dass Sie eine beliebige Anzahl von iterierbaren Argumenten verwenden können, nicht nur zwei. 

zip(data, sorted(data))

Es ist nicht notwendig, einer Variablen einen Wert zuzuweisen, wenn Sie die Variable nicht mehr als einmal verwenden werden. Iterieren Sie nun über Wertepaare mit der Schleife for. 

for i, j in zip(data, sorted(data))
Erklären Sie, was ich mit den Paaren tun soll

Sie müssen sie nur vergleichen und einen logischen Wert für ihre Ungleichheit erhalten. 

i!= j for i, j in zip(data, sorted(data))
Wie kann man die Anzahl der ungleichen Elemente zählen?

Python kann logische Werte als ganze Zahlen interpretieren: True ist gleich 1 und False ist gleich 0. Für jeden Fall von ungleichen Elementen erhalten Sie also i!= j => True => 1. Sie müssen nur den Ausdruck aus dem vorherigen Hinweis als Argument der eingebauten Funktion sum() verwenden. Sie akkumuliert 1-s, während 0-s keine Auswirkungen haben. 

sum(i!= j for i, j in zip(data, sorted(data)))
Wie sollte also meine endgültige Lösung aussehen?

Sie müssen nur einen endgültigen Wert zurückgeben und Ihren Code in einen Funktionsausdruck verpacken. Die Verwendung von Typ-Hinweisen für Eingabe- und Ausgabedaten wird dringend empfohlen.

Hier ist die gesamte Lösung:

def not_order(data: list[int]) -> int:

    return sum(i!= j for i, j in zip(data, sorted(data)))

Die Lösung ist dieselbe, aber mit der anonymen Funktion lambda, dem Standardmodul operator und der eingebauten Funktion map(). 

from operator import ne


not_order = lambda data: sum(map(ne, data, sorted(data)))
