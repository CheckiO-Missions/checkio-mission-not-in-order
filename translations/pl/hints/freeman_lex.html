Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Proszę złapać jakiś pomysł 💡! Proszę utworzyć uporządkowaną wersję podanego data i porównać elementy o tym samym indeksie. Proszę policzyć elementy, które nie są równe.

Ok, od czego powinienem zacząć?

Aby uzyskać nową listę, która jest posortowanym wariantem innej listy, proszę użyć wbudowanej funkcji sorted(). Kolejność rosnąca według wartości jest domyślna, więc nie trzeba zmieniać żadnego innego argumentu. 

data1 = sorted(data)
Co robić dalej?

Teraz proszę wykonać iterację po obu listach jednocześnie, indeks po indeksie. Idealnym narzędziem do tego zadania jest wbudowana funkcja zip(). Tworzy ona generator, który zwraca wartości o tym samym indeksie. Proszę zauważyć, że można użyć dowolnej liczby argumentów iterowalnych, nie tylko dwóch. 

zip(data, sorted(data))

Nie ma potrzeby przypisywania wartości do zmiennej, jeśli nie zamierzamy używać zmiennej więcej niż jeden raz. Teraz proszę iterować po parach wartości za pomocą pętli for. 

for i, j in zip(data, sorted(data))
Proszę wyjaśnić, co należy zrobić z parami?

Musi je Pan po prostu porównać i uzyskać wartość logiczną ich nierówności. 

i!= j for i, j in zip(data, sorted(data))
Jak policzyć liczbę nierównych elementów?

Python potrafi interpretować wartości logiczne jako liczby całkowite: True równa się 1, a False równa się 0. Tak więc każdy przypadek nierównych elementów daje i!= j => True => 1. Wystarczy użyć wyrażenia z poprzedniej podpowiedzi jako argumentu wbudowanej funkcji sum(). Kumuluje ona 1-s, podczas gdy 0-s nie ma żadnego efektu. 

sum(i!= j for i, j in zip(data, sorted(data)))
Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy zwrócić wartość końcową i zawinąć kod w wyrażenie funkcyjne. Używanie podpowiedzi typu dla danych wejściowych i wyjściowych jest wysoce zalecane.

Oto całe rozwiązanie:

def not_order(data: list[int]) -> int:

    return sum(i!= j for i, j in zip(data, sorted(data)))

Rozwiązanie jest takie samo, ale z użyciem anonimowej funkcji lambda, standardowego modułu operator i wbudowanej funkcji map(). 

from operator import ne


not_order = lambda data: sum(map(ne, data, sorted(data)))
